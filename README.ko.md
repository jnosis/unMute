## Features

- 현재 탭 음소거/음소거 해제
- 현재/최근 소리나는/지정된 탭 제외하고 모든 탭 자동 음소거
- 항상 모든 탭 음소거
- 모든 탭 음소거 토글
- 사용자 버튼 설정
